Project Insights

Soda a los Chizos:
This project started as an idea to capture the essence of a typical Costa Rican "soda" (small local restaurant). I focused on a warm and cozy visual design using HTML and CSS to reflect the local food culture. The layout was designed to be intuitive, friendly, and visually appealing.

Supermarket System:
A web system I developed as part of my academic work, focused on managing products, shopping carts, and order records. I used PHP for server-side logic and MySQL as the database, all integrated in XAMPP. I learned to handle user sessions and build full CRUD functionalities.

Panadería Dulce Trigo:
One of my most complete and ambitious projects. I combined Oracle and PHP within XAMPP to create a functional bakery website where users can register, log in, browse products, and place orders. The main challenge was establishing smooth communication between Oracle and the web environment.

Tú Outfit CR:
An online clothing store built in Visual Studio 2022 using C#. The website was created with the .NET MVC framework and integrates MySQL for database management. I also added Power BI to generate product and sales reports for business insights.

Perfumería El Guarco:
A web application designed for a local perfume store. I used HTML, CSS, PHP, and Power BI to build the system and present business data visually. This project helped me sharpen my UI design skills and data integration techniques for client-facing websites.
